# Задача 3

## Что требуется

Реализовать консольную программу для нормализации лог-файлов из двух входных форматов в единый выходной формат.

## План структуры

- `src/Cleverence.TestTask.Core/Task3`
- `tests/Cleverence.TestTask.Tests/Task3`

## Архитектурный фокус

- разделить парсинг форматов;
- централизовать нормализацию записи;
- отдельно обрабатывать невалидные строки;
- сделать удобный сценарий запуска через `Console`.

## Принятое допущение

В тексте ТЗ формат даты в выходе указан как `DD-MM-YYYY`, но примеры выходных строк показывают формат `YYYY-MM-DD`.

В текущем решении принят формат из примеров:

- `2025-03-10`

## Статус

Реализованы парсеры входных форматов и нормализатор записи. Файловый процессор еще не реализован.
