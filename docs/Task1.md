# Задача 1

## Что требуется

Реализовать алгоритмы сжатия и декомпрессии строки, состоящей из маленьких букв латинского алфавита.

## Где находится код

- `src/Cleverence.TestTask.Core/Task1`
- `tests/Cleverence.TestTask.Tests/Task1`

## Принятые правила

- исходная строка для сжатия должна содержать только символы `a-z`;
- одиночный символ хранится без счетчика;
- запись вида `a1` считается невалидной, потому что нарушает формат задачи;
- запись вида `a0` и формы с ведущим нулем тоже считаются невалидными.

## Как проверять

1. Выполнить `dotnet test tests/Cleverence.TestTask.Tests/Cleverence.TestTask.Tests.csproj --no-restore`.
2. Убедиться, что проходят тесты из папки `Task1`.

## Что уже покрыто тестами

- пример из ТЗ;
- одиночные символы;
- невалидные входные данные;
- round-trip сценарии `Compress -> Decompress`.
